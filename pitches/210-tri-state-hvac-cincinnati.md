# Tri-State HVAC Solutions — Cincinnati, OH

## Company Overview
- **Industry:** Construction / Mechanical Contracting
- **Location:** Cincinnati, Ohio
- **Estimated Size:** 100-140 employees
- **Type:** Commercial HVAC installation and service contractor

## Pain Points
- Service call scheduling is reactive — no predictive maintenance offerings
- Technician dispatching doesn't account for skills, parts availability, or location
- Estimating large commercial installations takes weeks
- Customer contracts and warranty tracking in filing cabinets

## Proposed AI Solution
Predictive maintenance platform using building sensor data to identify HVAC issues before failures, enabling proactive service contracts. AI-optimized technician dispatch matching skills, truck inventory, and proximity. Automated estimating tool for commercial installations using building specs and historical project data.

## Value Proposition
- Create new recurring revenue stream with predictive maintenance contracts
- 20% more service calls per day with optimized dispatch
- Cut commercial estimating time by 50%
